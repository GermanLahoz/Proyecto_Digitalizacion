# Mi Portfolio Web Avanzada

## Descripción
Este proyecto es una página web personal con un diseño moderno, animaciones, y formulario funcional que guarda datos en Firebase Firestore. Es responsive y tiene una barra de navegación fija.

## Tecnologías usadas
- HTML
- Firebase Firestore para la base de datos
- Firebase Hosting (o el servicio que uses) para desplegar

## Funcionalidades principales
- Barra de navegación fija y responsiva.
- Secciones con animaciones al hacer scroll.
- Formulario de contacto con validación.
- Guardado de mensajes en Firestore.

## Cómo usar
- Abre el sitio web en un navegador.
- Completa el formulario de contacto.
- Los datos se guardan en la base de datos Firebase.
